# aml-bigdata-anomaly-detection

Big Data System for Suspicious Money Laundering Detection using Isolation Forest

## Description

This project implements a Big Data-based system for detecting suspicious Anti-Money Laundering (AML) transactions using Machine Learning anomaly detection techniques.

The system was developed as an undergraduate Software Engineering thesis.
